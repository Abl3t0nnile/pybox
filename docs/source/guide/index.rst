Guide
=====
Dieser Guide bietet eine schrittweise Einführung in die grundlegenden Funktionen der PyBox Engine.

Vorwort
-------
Aus einer abstrakten Sicht, besteht die PyBox Engine aus zwei grundlegenden Komponenten. Einer Datenbank, die persistente Objekte enthält, sowie einer interaktiven Laufzeitumgebung, mit deren Hilfe die Objekte der Datenbank adressiert, interpretiert und modifiziert werden können. Die Datenbank steht dabei für die Spielwelt und ihren gegenwärtigen Zustand. Die Laufzeitumgebung bietet das Grundgerüst.
